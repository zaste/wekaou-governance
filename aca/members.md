## How to Become a Member

### For Individuals

1. Contribute to WEKAOU repositories (see eligibility in `statutes.md`)
2. After meeting criteria, open an issue in this repository: **"Membership Application: [Your Name]"**
3. Include:
   - GitHub username
   - Brief introduction
   - Links to 3+ contributions or RFC participations
4. TSC reviews quarterly and approves via simple majority

### For Organizations

1. Review membership tiers in `statutes.md`
2. Contact Steering Committee via governance repository issue
3. Provide:
   - Organization details
   - Intended membership tier
   - Use case or implementation details
4. Steering Committee negotiates terms and submits for TSC approval
